# @dotdo/client

## 0.3.0

### Minor Changes

- Initial public release to npm

  ### @dotdo/client v0.3.0
  - Cap'n Web RPC client SDK for Durable Objects
  - Automatic promise pipelining for efficient RPC batching
  - WebSocket transport with reconnection handling
  - TypeScript-first with full type inference

  ### @dotdo/react v0.2.0
  - React hooks and components for dotdo Durable Objects
  - Real-time sync with `useDO` and `useLiveQuery` hooks
  - TanStack Query integration via `@dotdo/react/tanstack`
  - Admin UI components for DO inspection
  - Optimistic updates and offline support
