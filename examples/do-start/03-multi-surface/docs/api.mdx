# API Reference

Every Durable Object exported from your surface files automatically gets REST endpoints.

## Workspace API

The `Workspace` DO is defined in `App.tsx` and available to all surfaces.

### Get Workspace

```bash
GET /Workspace/:id
```

Returns the workspace data and all stored things.

**Response:**
```json
{
  "$id": "acme",
  "$type": "Workspace",
  "things": [...]
}
```

### Create Project

```bash
POST /Workspace/:id/createProject
Content-Type: application/json

{
  "name": "My Project",
  "description": "Optional description"
}
```

Creates a new project in the workspace.

**Response:**
```json
{
  "$id": "proj_abc123",
  "$type": "Project",
  "name": "My Project",
  "description": "Optional description",
  "status": "active",
  "createdAt": "2026-01-13T12:00:00Z"
}
```

### List Projects

```bash
GET /Workspace/:id/listProjects
```

Returns all projects in the workspace.

**Response:**
```json
[
  {
    "$id": "proj_abc123",
    "$type": "Project",
    "name": "My Project",
    "status": "active"
  }
]
```

### Archive Project

```bash
POST /Workspace/:id/archiveProject
Content-Type: application/json

{
  "id": "proj_abc123"
}
```

Archives a project by setting its status to "archived".

## Things API

The underlying Things store provides standard CRUD operations:

```bash
# Create
POST /Workspace/:id/things
{ "$type": "Customer", "name": "Alice" }

# Read
GET /Workspace/:id/things/:thingId

# Update
PATCH /Workspace/:id/things/:thingId
{ "name": "Alice Smith" }

# Delete
DELETE /Workspace/:id/things/:thingId

# List by type
GET /Workspace/:id/things?$type=Customer
```

## Adding More Endpoints

Export additional methods from your DO class:

```tsx
export class Workspace extends DO {
  // Each public method becomes an endpoint
  async searchProjects(query: string) {
    const all = await this.things.list({ $type: 'Project' })
    return all.filter(p => p.name.includes(query))
  }

  async getProjectStats() {
    const projects = await this.things.list({ $type: 'Project' })
    return {
      total: projects.length,
      active: projects.filter(p => p.status === 'active').length,
      archived: projects.filter(p => p.status === 'archived').length,
    }
  }
}
```

Available at:
- `GET /Workspace/:id/searchProjects?query=foo`
- `GET /Workspace/:id/getProjectStats`
