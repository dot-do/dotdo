# UI

dotdo uses MDXUI for building interfaces with MDX components, integrated with TanStack Start.

## MDXUI Component Libraries

### Beacon
Marketing sites, landing pages, and public-facing content:

```mdx
import { Hero, Features, Pricing } from 'mdxui/beacon'

<Hero
  title="Build your 1-Person Unicorn"
  subtitle="Business-as-Code for autonomous businesses"
  cta={{ text: "Get Started", href: "/signup" }}
/>

<Features items={features} />

<Pricing plans={plans} />
```

### Cockpit
Admin dashboards, internal tools, and applications:

```mdx
import { Dashboard, DataTable, Chart } from 'mdxui/cockpit'

<Dashboard>
  <Chart data={revenueData} type="line" />
  <DataTable
    data={customers}
    columns={['name', 'email', 'plan', 'mrr']}
  />
</Dashboard>
```

## TanStack Integration

### TanStack Start

```typescript
import { createFileRoute } from '@tanstack/react-router'
import { MDXContent } from 'mdxui'

export const Route = createFileRoute('/docs/$')({
  component: DocsPage,
  loader: async ({ params }) => {
    return loadMDX(params._splat)
  }
})

function DocsPage() {
  const { content } = Route.useLoaderData()
  return <MDXContent content={content} />
}
```

### TanStack Query

```typescript
import { useQuery } from '@tanstack/react-query'

function CustomerList() {
  const { data } = useQuery({
    queryKey: ['customers'],
    queryFn: () => $.Customer.list()
  })
  return <DataTable data={data} />
}
```
