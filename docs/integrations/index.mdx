# Integrations

Individual integration guides for specific services. Each page provides detailed documentation for a single service SDK.

## How Integrations Work

dotdo provides two levels of documentation:

1. **Compat Categories** (`/docs/compat/`) - Overview pages that explain how groups of similar SDKs work (databases, messaging, search, vector, realtime, graph)
2. **Individual Integrations** (`/docs/integrations/`) - Detailed guides for specific services

Each integration is a drop-in replacement for the official SDK, backed by Durable Objects for edge-native performance:

- **Familiar APIs** - Use the same methods and patterns you already know
- **Edge-native** - Runs on Cloudflare's global network (300+ cities)
- **DO-backed storage** - Persistent state without connection pooling issues
- **Zero vendor lock-in** - Switch back to official SDKs anytime

## Installation

Install integrations via npm:

```bash
npm install @dotdo/stripe
npm install @dotdo/algolia
npm install @dotdo/pusher
```

Configure in your `wrangler.jsonc`:

```json
{
  "vars": {
    "STRIPE_SECRET_KEY": "sk_...",
    "ALGOLIA_APP_ID": "...",
    "ALGOLIA_API_KEY": "..."
  }
}
```

## Quick Start

All integrations follow the compat SDK pattern for API compatibility:

```typescript
import { stripe } from '@dotdo/stripe'

const customer = await stripe.customers.create({
  email: 'customer@example.com'
})
```

```typescript
import { algoliasearch } from '@dotdo/algolia'

const client = algoliasearch('APP_ID', 'API_KEY')
const { results } = await client.search({
  requests: [{ indexName: 'products', query: 'laptop' }]
})
```

```typescript
import Pusher from '@dotdo/pusher'

const pusher = new Pusher({ appId: '...', key: '...', secret: '...' })
await pusher.trigger('my-channel', 'my-event', { message: 'hello' })
```
