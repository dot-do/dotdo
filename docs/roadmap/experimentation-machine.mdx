# Experimentation Machine

The Experimentation Machine helps you find product-market fit through systematic testing. Based on Jeff Bussgang's framework, it treats AI as a 10x accelerant for running and learning from experiments.

<Callout type="info">
Finding PMF is often luck. The Experimentation Machine makes it systematic - you either find fit or learn why faster.
</Callout>

## The HUNCH Metrics Framework

Product-market fit isn't a binary state - it's measurable. The HUNCH framework gives you concrete signals:

| Metric | Question | PMF Signal | How to Measure |
|--------|----------|------------|----------------|
| **H**air-on-Fire | Is this a must-have? | >40% "very disappointed" | Sean Ellis test |
| **U**sage | Are they using it? | >40% weekly active | DAU/WAU ratio |
| **N**PS | Would they recommend? | >50 NPS | Survey |
| **C**hurn | Are they staying? | <5% monthly | Cohort analysis |
| **H**LTV/CAC | Is it profitable? | >3x ratio | Unit economics |

### Measuring HUNCH in Code

```typescript
// Configure HUNCH tracking for your business
const pmf = await $.measure({
  hairOnFire: metrics.seanEllis,     // % who'd be "very disappointed"
  usage: metrics.weeklyActive,       // WAU/MAU ratio
  nps: metrics.netPromoterScore,     // Survey results
  churn: metrics.monthlyChurn,       // Monthly churn rate
  ltv_cac: metrics.lifetimeValue / metrics.acquisitionCost,
})

// Check PMF status
if (pmf.score > 0.7) {
  console.log('Strong PMF signal - ready to scale')
} else if (pmf.score > 0.4) {
  console.log('Weak PMF - more experimentation needed')
} else {
  console.log('No PMF - pivot or iterate on hypothesis')
}
```

## Running Experiments

The Experimentation Machine provides infrastructure for systematic A/B testing:

### Experiment API

```typescript
// Define an experiment
const pricingTest = await $.experiment('pricing-2024-q1', {
  hypothesis: '$499/year converts better than $299/year for serious users',
  variants: [
    { id: 'control', value: 299, description: 'Current price' },
    { id: 'premium', value: 499, description: 'Higher price point' },
    { id: 'enterprise', value: 799, description: 'Enterprise positioning' },
  ],
  metric: 'conversion_rate',
  secondaryMetrics: ['revenue_per_visitor', 'trial_completion'],
  sampleSize: 1000,
  confidence: 0.95,
  duration: '2 weeks',
})

// Get variant for a user
const variant = await pricingTest.getVariant(userId)

// Track conversion
await pricingTest.track(userId, 'converted', { revenue: variant.value })

// Check results
const results = await pricingTest.results()
// { winner: 'premium', confidence: 0.97, lift: '+23%' }
```

## Learning Loops

The real power of the Experimentation Machine is **learning loops** - insights that compound across experiments.

### Pooled Insights

Experiments generate insights that apply across your entire business:


## Hypothesis Validation Workflow

The complete flow from hypothesis to validated learning:

```typescript
export class TaxPrepAI extends Startup {
  async validate() {
    // 1. State your hypothesis
    const hypothesis = await $.hypothesis({
      assumption: 'Freelancers will pay $500/year for AI tax prep',
      metric: 'conversion_rate',
      threshold: 0.03,  // 3% conversion is viable
    })

    // 2. Design the experiment
    const experiment = await hypothesis.design({
      method: 'landing-page',
      variants: ['$299', '$499', '$699'],
      traffic: 1000,
    })

    // 3. Run the experiment
    await experiment.run()

    // 4. Analyze results
    const results = await experiment.analyze()

    // 5. Learn and iterate
    if (results.validated) {
      await $.log.learning('Price point validated at $499')
      return { next: 'test-feature-set' }
    } else {
      await $.log.learning(`Price point failed: ${results.insight}`)
      return { next: 'test-lower-price' }
    }
  }
}
```

## From Experimentation to Autonomous Business

Once HUNCH metrics signal PMF, you're ready to scale:

```typescript
async function checkReadiness() {
  const hunch = await $.measure.hunch()

  if (
    hunch.hairOnFire > 0.4 &&
    hunch.usage > 0.4 &&
    hunch.nps > 50 &&
    hunch.churn < 0.05 &&
    hunch.ltv_cac > 3
  ) {
    console.log('PMF achieved - ready for Autonomous Business phase')
    return true
  }

  console.log('Continue experimenting:', hunch.weakest)
  return false
}
```

When all signals are green, move to the [Autonomous Business](/docs/roadmap/autonomous-business) phase where AI agents take over operations and you scale.
