# Actions

Actions are reusable, typed operations that can be composed and executed across your application.

## Defining Actions

```typescript
import { Action } from 'dotdo'

export const sendWelcomeEmail = Action({
  name: 'send-welcome-email',
  input: {
    userId: 'string',
    email: 'string'
  },
  handler: async ({ userId, email }, ctx) => {
    await ctx.email.send({
      to: email,
      template: 'welcome',
      data: { userId }
    })
  }
})
```

## Action Middleware

```typescript
import { Action, withRetry, withTimeout } from 'dotdo'

export const processPayment = Action({
  name: 'process-payment',
  middleware: [
    withRetry({ maxAttempts: 3 }),
    withTimeout({ ms: 30000 })
  ],
  input: { amount: 'number', customerId: 'string' },
  handler: async ({ amount, customerId }) => {
    // Payment processing logic
  }
})
```

## Action Composition

```typescript
import { compose } from 'dotdo'

export const onboardCustomer = compose(
  createAccount,
  sendWelcomeEmail,
  setupBilling,
  notifySales
)
```
